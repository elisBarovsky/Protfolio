Welcome to my interactive digital headquarters. This portfolio is a blend of clean web architecture and immersive game-inspired design.

## 🕹️ The Experience
This isn't just a static site. It's a "Character Sheet" of my professional journey, featuring:
* **Custom CRT/Neon Aesthetics:** A cyberpunk-inspired UI.
* **Interactive Skills Grid:** Explore my tech stack through a gamified inventory.
* **Live Projects:** Links to my published games on itch.io.
* **Contact System:** Integrated form to transmit messages directly to my HQ.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Custom Properties & Animations), JavaScript (ES6+).
* **Visuals:** Canvas API for the hero background, FontAwesome for icons.
* **Deployment:** GitHub Pages.
* **Communication:** Formspree API.
